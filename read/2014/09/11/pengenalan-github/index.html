Pengenalan Github

Sep 11th, 2014 1:14 pm

Assalamualaikum wr.wb Akhirnya setelah sekian lama tidak menulis lagi di halaman ini, aku ada waktu yaa setidaknya menulis dan berbagi ilmu. Judul kali ini adalah “Pengenalan Github”. Karena sesuai dengan pepatah mengatakan tak kenal maka tak sayang, begitu pun dengan ilmu apabila temen ingin mempelajari ilmu temen pun harus mengetahui ilmu itu berasal dari mana atau lebih jelasnya pengertian atau deskripsi. Langsung tkp saja yuk.

Ada yang tahu github ? pasti sebagian temen-temen ada yg sudah mengenal github. Bagi yang sudah jangan sombong cobalah untuk berbagi ilmu toh ilmu kalian tidak akan habis jika berbagi. Yaa github merupakan software hosting or bahasa kitanya wadah software untuk proyek open source yang menggunakan tools sistem revisi control git. Nah loh git apa lagi yaa, git adalah tool untuk melakukan revisi code, sedangkan github adalah webhostingnya. Mudahnya Github adalah Webhosting untuk proyek proyek software seperti Google code. Github adalah jejaring social untuk software developer.

Perbedaan Github dengan Project hosting yang lain adalah sebagai berikut.

  • Github menggunakan Git sebagai tool utama dalam merevisi code.
  • Github menyediakan free hosting untuk proyek open source, namun Github juga menawarkan hosting berbayar untuk proyek perusahaan atau pribadi yang sifatnya privat.
  • Github ditargetkan untuk para developer yang bekerja secara tim dan tidak disatu tempat.
  • Github mempunyai fitur Social seperti Twitter seperti follow atau favorit (Star).
  • Github mendukung semua bahasa pemrograman, anda bebas memakai bahasa dan tool yang biasa anda pakai.

Fitur-fitur yang terdapat pada github.

  • Github menggunakan profile pengguna (user profile).
  • Kita bisa mengikuti user lain (follow) or sebaliknya.
  • Watch memonitor repository tertentu. Setiap ada perubahan di Repo, maka kita akan dikirim notifikasi.
  • Fork mengcopy keseluruhan code repository ke repo kita sendiri.
  • Star fungsinya sama dengan Bookmark.

User Profile Github

Profile dari github sama dengan jejaring sosial twitter maupun facebook. Profilenya berisi foto jumlah repository jumlah follower jumlah user yang kita ikuti (following) dan star. Neh aku kasih contoh pictnya.

heemm biasanya si jika followernya banyak kemungkinan besar bahwa temen itu mahir skillnya. Contoh aja aku masih sedikit followernya berarti aku masih newbie.

Fitur searching Github memiliki lebih dari 50 juta repository sofware. Nah banyak banget kan? Jika temen ingin mencari proyek open source tertentu, gunakan fitur pencarian.

Struktur halaman di github

Contoh di gambar 2 adalah struktur poyek dari Twitter Bootstrap, CSS framework paling populer saat ini. Mari kita analisa satu persatu struktur halamannya.

Bagian Statistik

Bagian pertama adalah statistik, kia bisa melihat jumlah Watch, star dan fork dibagian atas halaman. Bootstrap adalah salahsatu proyek paling populer di github, maka jangan heran jika ada lebih dari 63 ribu user yang memberi tanda bintang.

Bagian Code

Bagian utama dari struktur proyek ini adalah code itu sendiri. Kode ini juga berisi penjelasan kapan dan apa file terakhir yang diedit berserta penjelasan perubahan apayang terjadi.

Bagian Clone dan Download

Clone in desktop di pakai untuk mengkopi source code proyek terserbut ke komputer kita , namun komputer kita harus sudah terinstall Github for Windows atau Mac. Bagian paling penting dan paling sering dipakai adalah menu DOWNLOAD. jika temen ingin langsung mencoba code di komputer , cukup klik download, maka source code otomatis akan dizip dan diunduh ke kompuer temen.
