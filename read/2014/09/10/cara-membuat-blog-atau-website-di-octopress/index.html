Cara Membuat Blog Atau Website Di Octopress

Sep 10th, 2014 7:53 pm

Octopress merupakan sebuah framework yang dirancang oleh Brandon Mathis. Framework ini merupakan mesin blog atau wesite secara statis atau lebih tepatnya html statis. Dengan menggunakan framework ini (Octopress) kita hanya perlu mengclone atau menggandakan dari github. Bagaimana cara menginstallnya, yuk disimak satu persatu caranya dimulai dari

  • Teman-teman install github terlebih dahulu. Cara install github bisa cek disini.

  • Install ruby. Cara install ruby dapat cek disini.

  • Apabila semuanya sudah terinstall saatnya mengclone git dari repos octopressnya di terminal (ctrl+alt+t) ubuntu

  • Ketikan:
  • Lalu install gem
  • Install bundle
  • Dan yang terakhir install
  • Temen-temen bisa mencoba mengetest dengan cara membuat post di octopress, bagaimana caranya tunggu kelanjutan ilmunya. Terima kasih sudah mampir, assalamualaikum.
