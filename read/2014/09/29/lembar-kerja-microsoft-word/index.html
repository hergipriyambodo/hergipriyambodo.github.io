Lembar Kerja Microsoft Word

Sep 29th, 2014 4:02 pm

Tanpa basa-basi lagi yuk kita mulai dari membuka file microsoft word. Apabila temen-temen tidak tahu cara membuka microsof word, sini aku kasih tahu caranya. Lebih lengkapnya cek pict dibawah ini

temen-temen tinggal klik yang sudah saya kasih bulatan/temen-temen bisa mengikuti arahan dari aku.

  • Pilih menu start atau pilih tombol yang bergambar logo windows pada keyboard

  • Pilh dan klik microsoft windows pada menu bar

  • Tunggu sampai jendela microsoft word terbuka

  • Apabila telah terbuka maka tampilannya akan seperti ini

Oh ya untuk penjelasan angka-angka yang tertera pada gambar bisa dicek dibawah ini, cekidot vroh. Keterangannya

1.Ribbon menu

Adalah menu utama untuk lembar kerja MS Word. Beberapa orang masih sering menyebutnya dengan Menu Home. Di menu ini semua tools dan command diletakkan. Menu utama ini juga memiliki beberapa bagian atau sub-sub terkait dengan tab menunya.

2.Lembar kerja doc

Visualisasi kertas tempat kita mengetikkan dan memasukkan gambar di dalam dokumen Word.

3.Toolbar

Sama seperti ribbon menu yaa vroh.

4.Nama File

Nama file yang sedang terbuka/aktif. Secara default, ketika membuka MS Word maka file akan bernama Document1 dengan eksistensi file .docx. Nama file tersebut bisa kita ubah dan sesuaikan sesuai dengan keinginan.

5.Ruler

Penggaris sebagai pembantu dalam mengukur margin, kertas, dan juga jarak antar paragraf. Sepertinya semua orang tahu, apa gunanya penggaris itu.

6.Display

Temen-temen bisa mengatur Display atau tampilan dari bentuk lembar kerja kita dengan beberapa pilihan yang tersedia. Lebih jelasnya silahkan baca artikel Tampilan Halaman MS Word.

7.Tab Menu

Bagian dari Menu Ribbon, setiap perpindahan sub menu sering diberi istilah tab menu. Seperti, tab menu Home, tab Menu Page Layout, dan lain-lain.

8.Vertical Sliding Bar

Alat bantu untuk menggulir/scroll halaman lembar kerja kita. Jika kita menggunakan display yang lebih besar, maka akan muncul Horizontal Sliding bar di bagian bawah areal kerja untuk membantu kita menggeser ke kanan dan kiri halaman tersebut.

Ya seginilah sekiranya penjelasan mengenai area kerja microsoft word, apabila ada yang kurang atau sekedar bertanya jangan ragu untuk menghubungi aku via email. Terima kasih *kecup
