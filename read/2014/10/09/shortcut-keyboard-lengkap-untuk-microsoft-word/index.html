Shortcut Keyboard Lengkap Untuk Microsoft Word

Oct 9th, 2014 7:15 pm

Shortcut adalah kombinasi-kombinasi tombol keyboard untuk melakukan sebuah perintah (command line) di MS Word tanpa harus masuk ke menu utama perintah tersebut. Dengan menggunakan shortcut ini, temen-temen bisa meringkas waktu untuk mengetik dan mengedit lembar kerja. Saya mencoba mengumpulkan Shortcut lengkap dari tombol A-Z serta fungsi-fungsinya. Silahkan lihat list shorcut lengkap dibawah ini. Shortcut ini Berlaku untuk MS Word 2007 ke atas.

Tombol Shortcut Fungsi
ctrl + A Memilih/blok semua teks
ctrl + B Menebalkan huruf (bold)
ctrl + C Menggandakan kata (copy)
ctrl + D Memilih jenis huruf (font)
ctrl + E Rata tengah (center)
ctrl + F Mencari kalimat (find)
ctrl + G Pergi ke (go to)
ctrl + H Find and Replace
ctrl + I Membuat cetak miring (italic)
ctrl + J Rata kiri kanan (justify)
ctrl + K Input alamat tautan (insert hyperlink)
ctrl + L Rata kiri (left)
ctrl + M Menggeser penempatan paragraf ke kanan
ctrl + N Membuat lembar kerja baru
ctrl + O Membuka file yang tersimpan
ctrl + P Mencetak dokumen
ctrl + Q Format paragraf yang telah dibuat
ctrl + R Rata kanan (right)
ctrl + S Menyimpan dokumen (save)
ctrl + T Menggeser hangin indent bawah ke kanan)
ctrl + U Membuat garis bawah (underline)
ctrl + V Tempel (Paste)
ctrl + W Keluar dari dokumen
ctrl + X Memotong/pindah (cut)
ctrl + Y Mengembalikan perintah yang dibatalkan (redo)
ctrl + Z Membatalkan perintah sebelumnya (undo)

Tambahan selain dari huruf A-Z, aku juga akan memberikan tambahan jalan pintas.

Tombol Shortcut Fungsi
ctrl + Shift + M Menggeser penempatan paragraf ke kiri
ctrl + Shift + T Menggeser hangin indent bawah ke kiri
ctrl + [ Mengecilkan huruf
ctrl + ] Membesarkan huruf
shift + F3 Pengaturan Case : Uppercase, Lowercase, Sentence Case
Alt + F4 Keluar dari lembar kerja
ctrl + Enter Melompat ke halaman baru
F12 Menyimpan kembali dengan nama yang sama
